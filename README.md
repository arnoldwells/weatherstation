# weatherstation
display weather forecast from openweathermap.org on Pimorini Inky Impression eink display, send data to io.adafruit.com, generate weather image from getimg.ai and store it on google drive
![sped up animation](/sped-up-animation.gif)

#### Further improvements and features
- Query the google calendar and add a countdown for the next two birthday events
- Show the current moon phase
- Create more weather icons
- Experiment with other colour options on the eink display
- Truncate weather summary if it gets too long
- Use another (less pessimistic) source for weather forcast data
- Detect a summary containing an error message and don't use it for getimg
